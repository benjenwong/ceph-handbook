# 简介
---

《Ceph 运维手册》 汇总了 Ceph 在使用中常见的运维和操作问题， 主要用于指导运维人
员的相关工作。

## 环境

本手册是基于以下环境

- CentOS 7.5， Ceph Luminous版。
